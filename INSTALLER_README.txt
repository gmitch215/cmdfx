-- CmdFX Installer --

Welcome to the CmdFX installer. This will help you get started with using the cmdfx game engine, designed to be used in your terminal.

Prerequisites: cmdfx renders through a curses library. On Linux install ncurses
(libncursesw5-dev) and the ALSA sound library (libasound2-dev); macOS ships
ncurses already. On Windows, install ncurses via MSYS2
(pacman -S mingw-w64-x86_64-ncurses) or PDCurses -- if none is found, the build
fetches PDCurses automatically.

There's not much to explain. Just go ahead and keep moving. There's some helpful links below.

GitHub: https://github.com/gmitch215/cmdfx
Documentation: https://gmitch215.github.io/cmdfx
