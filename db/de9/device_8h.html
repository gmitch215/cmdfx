int Device_getKeyboardKeyPressed ( )

Gets the current key being pressed, or -1 if no key is being pressed.

This is a non-blocking method. This method gets the current key being pressed on the keyboard. If no key is being pressed, it will return -1.

This method requires administrator privileges on Linux.

Returns
The key code of the key being pressed, or -1 if no key is being pressed.

int Device_getMouseButtonPressed ( )

Gets the current mouse button being pressed, or -1 if no button is being pressed.
