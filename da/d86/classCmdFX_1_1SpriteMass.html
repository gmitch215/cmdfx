Detailed Description

A C++ wrapper around a CmdFX_SpriteMass struct.

This class is a wrapper around the Sprite's mass declarations. It provides methods to get and set the mass of a sprite, as well as to reset it to its default value. The destructor will reset the mass when the object is destroyed.
